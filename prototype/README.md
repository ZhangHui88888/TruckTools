# TruckTools 原型图

## 卡车外贸综合工具平台 - MVP版本交互原型

### 📌 项目说明

本原型图基于 PRD 和技术架构文档设计，展示了 TruckTools 卡车外贸工具平台 MVP 版本的主要功能界面和交互流程。

### 🚀 如何查看原型

#### 方式一：直接打开 HTML 文件
直接在浏览器中打开 `index.html` 文件即可查看原型。

#### 方式二：使用本地服务器（推荐）
```bash
# 使用 Python 启动简单服务器
cd prototype
python -m http.server 8080

# 或使用 Node.js 的 serve
npx serve .
```

然后在浏览器中访问 `http://localhost:8080`

### 📋 原型包含的页面

#### 1. 登录页面
- 品牌展示区域
- 登录表单
- 记住登录/忘记密码

#### 2. 客户管理模块 ✅
- **名片识别与录入**：上传名片图片，AI自动识别
- **Excel批量导入**：通过Excel表格批量导入客户
- **客户信息表格**：管理客户信息，支持筛选排序导出

#### 3. 邮件营销模块 ✅
- **邮件模板管理**：创建和管理邮件模板
- **批量发送**：筛选客户，批量发送个性化邮件
- **发送日志**：查看发送历史和状态

#### 4. 系统设置 ✅
- 个人资料
- 账号安全
- SMTP邮箱配置

#### 5. 功能路线图
- 展示产品发展规划
- 各模块预计上线时间

#### 6. 即将推出页面
- 工作台（Dashboard）
- 销售管理
- 物流跟踪
- 供应链管理
- 文档中心
- 数据分析

### 🎨 设计特点

- **现代化UI**：采用清爽的蓝白配色，科技感与专业感并重
- **模块化导航**：侧边栏清晰展示所有模块，未开发功能标记"即将推出"
- **响应式设计**：适配不同屏幕尺寸
- **交互反馈**：hover效果、状态标签、进度展示
